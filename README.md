
# Dice Game

- This webpage features a two-player dice game that automatically rolls the dice upon refreshing. 
- The outcome results in either one player winning or a draw.
- Created this game while learning Document Object Model (DOM).



## Features

- Retro design
- Refresh the page to play.




## Tech Stack

- HTML5/CSS3
- JavaScript
- DOM
